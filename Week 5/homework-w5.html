These random graphs represent communication networks, so the biggest differences between them are connectivity and reliability. The more nodes are linked to each other, the higher the degree of the network is. So Network C has the highest degree, while network A has the lowest degree. In addition, the network C can transfer the data widely but has the problem of many to one that could affect some nodes. Network B has less links between nodes so it is better distributed than network C. Network A has the fewest links between nodes which can make it unreliable. For example, if the couples of links disconnect the node, it also disconnect from the network. So network C is the most reliable and network A is the most unreliable.
